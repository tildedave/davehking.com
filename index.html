Hi, I'm Dave King, a software professional living in Oakland, CA. Runner, backpacker, history buff.

{% markdown %} Software person in the Bay Area. I moved out here in 2014 after living in Blacksburg, Virginia for five years, which I had moved to after finishing up grad school at Penn State. I began programming when I was 8 writing text adventures in GWBASIC on my school's computer. The [day job](https://chartio.com/) tends to be some combination of JavaScript, Python, and Java - all objectively better technologies, all far less fun. In my spare time I backpack, read, and program. A favorite personal project is to write programs that play chess and I've written two so far: [one in C++](https://github.com/tildedave/apep-chess-engine) and [one in Golang](https://github.com/tildedave/ra-chess-engine). Over the last year I've been trying to learn more abstract algebra and number theory. {% endmarkdown %}
